## Traversals
### Breadth First Search (BFS)
- Queue (To store the child or connected nodes which still need to be explored).
- Unordered Map or vector (To find that a particular node is visited or not).
  
### Depth First Search (DFS)
Unordered Map or vector (To find that a particular node is visited or not).
